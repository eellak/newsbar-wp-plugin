## Ellak Newsbar Wordpress Plugin

Rewrite of legacy ellak newsbar iframe code to a Wordpress Plugin.

To use it add:

```
if( function_exists( 'ellak_newsbar' ) ) {
    ellak_newsbar();
} ?>
```

in the theme's `header.php` right after the initialization of the `<body>` tag.
